Imp log-normal

There is a note in the yaImpute paper about distances being assumed to be log-normal for all but RF. That is currently not relevant, but I would like to see the data transformed.

Summay

  • log(x + 1) is a good transformation if we decide the model from a distribution
  • It is unclear if imputing with transformed data was helpful
  • Changing the RF y variables reduced oobe by about .02. The highest one is most prev sp at .11.
  • The best rmsd is the one with diameters, the best oobe is the one with the new yrfs.
  • Running by tree instead of tpa scored better than by log(x+1) but worse than dia.
    • Probably best to convert to counts
  • RF w/ counts did better than eco in rmsd, but worse in oobe, everything was the same except the counts here.
  • There is code at the bottom for number of plots with regen plus etc for this data selection.

Notes

  • Scaling the tpa by the fia seedling adjustment factor of ~75, then transforming by log(x + 1) is a good transformation.
  • There may be an issue with using the St.Joe example RF y variables.
    • I think RF is using (y_r - y_t)^2 as part of the generalization error formula. So if those variables aren’t good, then the recursive check in RF will not produce the desired result.

RF Y variable importance

I think RF is using (y_r - y_t)^2 as part of the generalization error formula. So if those variables aren’t good, then the recursive check in RF will not produce the desired result. There is defineatly some sort of E - O test. I just can’t make certain exactly what it’s doing.

The current yrf variables are max spp, max sp tpa, and max tpa. The maximum species tpa seems to often also be the max tpa. I think it will be easier to test this than it would be to find the information in the reference material. I haven’t seen anything about it except in the St. Joe example.

Method randomForest works best when there are few variables and when factors are used rather than continuous variables. The whatsMax function is used to create a data frame of containing a list of the species of maximum basal area, and two other related variables.
